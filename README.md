Firmware used to power the Rep-Tile Oxford Maths Fest Project, built by Mahraz Ahmed and Vayun Khare on behalf of Professors David Kohan Marzagao and Josh Murphy for King's College London and the University of Oxford Mathematical Institute.

Custom library implementations are used! Please install the PN532.h library within Arduino IDE and update PN532.h and PN532.cpp with the custom implementations featured in this repository before writing any code.

Designed for the ESP32-WROOM1 module.
